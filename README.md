## Countries App  
  



### Proyecto individual para Henry Bootcamp  
Desarrollo de una web app sobre países donde podemos ver las banderas de los mismos, filtrar en orden alfabético, por cantidad de población, buscar por nombre.
También tiene la funcionalidad de asociar, crear y borrar actividades turísticas a cada país.

Para ver la app:
1. Forkear el repo
2. npm start en el back.
3. npm start en el front
4. enjoy!
